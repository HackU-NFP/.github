# SmartMint
 SmartMinst is the application which allows you to easily mint NFTs via LINE, a messaging app.

## Technology Stack
**Front-end**
- Next.js (TypeScript)

**Back-end**
- Go
- LINE Blockchain API
- LINE Messaging API
